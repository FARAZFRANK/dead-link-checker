Frank Dead Link Checker monitors your website for broken links, helping you fix 404 errors and redirect traffic to improve SEO and user experience.

== Description ==

**Frank Dead Link Checker** monitors your WordPress website for broken links and missing images. Keeping your site free of dead links is crucial for both SEO and user experience. This plugin helps you detect 404 errors and other link issues, allowing you to fix them quickly.

The plugin runs efficiently in the background, scanning your posts and pages for broken URLs. It provides a simple dashboard to view and manage these links without slowing down your site.

= Free Features =

*   **Automatic Scanning**: Automatically scans your **Posts** and **Pages** for broken links.
*   **Link Types**: Checks both Internal and External links.
*   **Gutenberg Support**: Fully compatible with the WordPress Block Editor.
*   **Visual Dashboard**: View reports of broken links directly in your WordPress admin.
*   **Smart Dismiss**: Dismiss specific links that you want the scanner to ignore.
*   **Manual Control**: Option to run manual scans when needed.
*   **Lightweight**: Designed to have minimal impact on your server performance.

= Why Use Frank Dead Link Checker? =

Broken links can harm your site's reputation and search engine rankings. Search engines like Google may penalize sites with excessive 404 errors. By using Frank Dead Link Checker, you can:

*   **Improve SEO**: Ensure search engines can crawl your site effectively.
*   **Better User Experience**: Prevent visitors from landing on error pages.
*   **Save Time**: Automate the process of link checking instead of doing it manually.

== Installation ==

1.  Upload the `frank-dead-link-checker` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **Link Checker** in your admin menu.
4.  Click **Scan Now** to start checking your site.

== Frequently Asked Questions ==

= Does this plugin slow down my website? =
No, Frank Dead Link Checker is designed to be lightweight. It limits the number of concurrent requests and runs properly in the background to ensure your site remains fast for visitors.

= Can I choose which links to scan? =
Yes, in the Free version, you can scan Posts and Pages. The Pro version allows you to include Custom Post Types, Comments, and specific page builder content.

= How do I fix a broken link? =
You can view the broken link in the dashboard and navigate to the specific post to edit it. The Pro version offers a "Redirect Manager" to fix links directly by setting up 301 redirects.

= What is the difference between Free and Pro? =
The Free version is perfect for blogs and standard websites needing basic link monitoring. The Pro version is built for power users, agencies, and large sites requiring detailed reports, faster scanning, redirect management, and page builder integration.

= Will it check links in Elementor or Divi? =
The Free version scans standard content. For deep scanning within Elementor, Divi, or WPBakery unique modules, you will need the Pro version.
